# Backlog Inicial --- Épicos e User Stories

Este documento apresenta a organização inicial do backlog do sistema,
estruturado por **Épicos** e **User Stories**.

------------------------------------------------------------------------

# Épico 1 --- Gestão de Usuários

**Descrição:**\
Este épico contempla todas as funcionalidades relacionadas ao cadastro,
autenticação e gerenciamento da conta dos usuários.

## User Stories

-   **US01 --- Cadastro de usuário**\
    *Como usuário, quero criar uma conta na plataforma para poder
    acessar o sistema.*

-   **US02 --- Realizar login**\
    *Como usuário, quero realizar login no sistema para acessar minhas
    funcionalidades.*

-   **US03 --- Encerrar sessão**\
    *Como usuário, quero realizar logout para encerrar minha sessão com
    segurança.*

------------------------------------------------------------------------

# Épico 2 --- Gerenciamento de Perfil

**Descrição:**\
Este épico envolve as funcionalidades relacionadas à atualização e
gerenciamento das informações pessoais do usuário.

## User Stories

-   **US10 --- Editar perfil**\
    *Como usuário, quero editar minhas informações de perfil para manter
    meus dados atualizados.*

-   **US11 --- Alterar foto de perfil**\
    *Como usuário, quero alterar minha foto de perfil para personalizar
    minha conta.*

-   **US12 --- Atualizar informações de contato**\
    *Como usuário, quero atualizar meu e-mail e número de telefone para
    manter meus dados corretos.*

------------------------------------------------------------------------

# Épico 3 --- Agendamento de Serviços

**Descrição:**\
Este épico contempla as funcionalidades relacionadas ao agendamento de
serviços na plataforma.

## User Stories

-   **US20 --- Criar agendamento**\
    *Como usuário, quero selecionar data e horário para agendar um
    serviço.*

-   **US21 --- Escolher serviço e pagamento**\
    *Como usuário, quero selecionar o serviço desejado e o método de
    pagamento no momento do agendamento.*

-   **US22 --- Confirmar agendamento**\
    *Como usuário, quero confirmar meu agendamento para garantir o
    horário selecionado.*

------------------------------------------------------------------------

# Épico 4 --- Gerenciamento de Agendamentos

**Descrição:**\
Este épico abrange funcionalidades de controle e gerenciamento dos
agendamentos realizados.

## User Stories

-   **US30 --- Visualizar agendamentos**\
    *Como usuário, quero visualizar meus agendamentos realizados para
    acompanhar meus compromissos.*

-   **US31 --- Cancelar agendamento**\
    *Como usuário, quero cancelar um agendamento caso não possa
    comparecer.*

-   **US32 --- Receber confirmação do agendamento**\
    *Como usuário, quero receber confirmação do agendamento para ter
    segurança de que foi registrado.*

------------------------------------------------------------------------

# Épico 5 --- Segurança da Conta

**Descrição:**\
Este épico trata das funcionalidades relacionadas à proteção e
recuperação de conta do usuário.

## User Stories

-   **US40 --- Solicitar redefinição de senha**\
    *Como usuário, quero solicitar redefinição de senha caso tenha
    esquecido minha senha.*

-   **US41 --- Alterar senha**\
    *Como usuário, quero alterar minha senha para manter minha conta
    segura.*

-   **US42 --- Receber confirmação de alteração de senha**\
    *Como usuário, quero receber confirmação da alteração de senha para
    garantir que a mudança foi realizada.*

------------------------------------------------------------------------

# Épico 6 --- Suporte Inteligente

**Descrição:**\
Este épico inclui funcionalidades relacionadas ao suporte ao usuário
através de inteligência artificial.

## User Stories

-   **US50 --- Utilizar assistente virtual**\
    *Como usuário, quero utilizar um assistente virtual para tirar
    dúvidas sobre o sistema.*

-   **US51 --- Receber orientações durante o uso do sistema**\
    *Como usuário, quero receber orientações da inteligência artificial
    para facilitar o uso da plataforma.*

-   **US52 --- Obter ajuda para realizar agendamentos**\
    *Como usuário, quero que a inteligência artificial me ajude a
    realizar agendamentos corretamente.*
