## Requisitos Funcionais (RF)

### RF01 — Cadastro de Usuarios
O sistema deve permitir que os usuários realizem o cadastro para posteriormente acessarem a plataforma.

### RF02 — Login
O sistema deve permitir que o usuário realize login, autenticando-se para acessar o sistema.

### RF03 — Editar o Perfil
O sistema deve permitir que o usuário edite as informações de seu perfil.
Exemplos:
- Alteração da foto de perfil.
- Alteração do numero de contato.
- Alteração do email de contato.
- Alteração do nome de Exibição.

### RF04 — Agendamento
O sistema deve permitir o usuario efetuar o agendamento selecionando:
- Data
- Horario
- Serviços
- Metodo de Pagamento

### RF05 — Alterar Senha
O sistema deve permitir que o usuário solicite a alteração de senha, caso tenha esquecido sua senha atual.

### RF06 - Logout
O sistema deve permitir que o usuário encerre sua sessão, desconectando-se da plataforma.

### RF07 - Cancelamento
O sistema deve permitir que o usuário cancele um agendamento previamente realizado.

### RF08 - Inteligencia Artificial
O sistema deve disponibilizar uma inteligência artificial para auxiliar o cliente durante a utilização da plataforma, fornecendo suporte e orientações.
